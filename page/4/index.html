配置sublime

#1. Package Control
使用sublime3最方便的安装package的方法就是使用Package Control，Package Control安装起来还是很容易的，在sublime的控制台中输入一句话（这句话看起来有点长）就好了，控制台的快捷键是Ctrl + ~，也可以从菜单View - Show Console打开，打开之后复制如下一句话（注意这真的是一句话）：

import urllib.request,os; pf = 'Package Control.sublime-package'; ipp = sublime.installed_packages_path(); urllib.request.install_opener( urllib.request.build_opener( urllib.request.ProxyHandler()) ); open(os.path.join(ipp, pf), 'wb').write(urllib.request.urlopen( 'http://sublime.wbond.net/' + pf.replace(' ','%20')).read())

如果发现这种简单的方法不能用的话可以试试手动安装的方法，方法连接在这里。

安装好之后可以打开sublime的命令模式（Ctrl + Shift + P），输入Package Control就可以看到Package Control的菜单了，可以在里面安装package，也可以删除，如果只是想安装package的话可以输入install作为关键字，回车之后就可以看到网络上的sublime3的各种package了。

#2. Python
当时买sublime的第一原因就是想要找个写python的IDE，然后配置一通下来之后发现sublime无愧于神器之称，充分的满足了程序员们想要敲命令的癖好……

##2.1 Anaconda
这里说的Anaconda不是那个python的集成科学计算库，而是sublime的一个package，使用Anaconda可以方便的进行命令补全，而且可以自动的给你的python代码进行pep8格式化，那么安装Anaconda非常简单，只需要在选择package control的安装功能后输入Anaconda进行搜索，然后点击全字匹配的那个就好了，package control会帮你把Anaconda安装好的。

然后的问题就是把Anaconda配置的让自己喜欢就好了。

自动检测文件夹改动情况

之前我并没有遇到其他Gitblog使用者遇到的每次发布新博客都要手动删除缓存的问题，原因是我的Gitblog根本无法产生缓存，每次访问都需要解析博客文件，访客少的时候自然没有问题，但是访客多了的话就有很大的隐患了。

所以我检查了Gitblog无法缓存的原因，发现是文件所有者出了问题，因为nginx服务器和php服务的owner是apache，而我之前把整个网站文件的owner改为了root，所以Gitblog无法产生缓存。

既然发现了问题就好办了，把文件的owner改回来就好了。

如何使用Git更新博客（一次失败的尝试）

开门见山，先来说一下我用git发布博文的整体结构。

  • 博客服务器一枚
  • Git@OSC帐号一枚
  • 本地电脑一台

发布工作的流程是这样的：

  1. 本地编写完成之后在博文目录提交并push到OSC的仓库
  2. 博客服务器从OSC的仓库中pull得最新博文。

这样博客服务器和本地电脑上面的git都是充当客户端的git，本来想看一看有没有直接上传的办法，找了一会没找到就采取这种办法了，不过这样也好，我的博文有四处存放点（本地+博客服务器+OSC仓库+坚果云）。

为了保证每次打开博客看到的都是最新的情况，我在GitBlog的index.php里面加了一句运行本地shell文件的命令，如下：
exec(".../gitpull.sh");
这样每次打开博客都会自动去远程仓库pull一下，总感觉pull频繁了会有安全隐患呢。。。先不管了，以后再说吧。其实刚才也尝试了crontab的方法（linux的定时器），设定的是每五分钟pull一次，但是好像没什么效果，加上这么做可能不能实时更新，就放弃了。以后的话看看改善一下gitpull这个脚本吧。

对了，我现在用的MarkDown是一个开源的Qt的本地编辑器CuteMarkEd，每次打开都要手动复制博文的前缀也是很麻烦的，所以最近想要在这个CuteMarkEd的基础上添加一些适合GitBlog的功能，这样用起来应该会方便一些吧。

但是Git的方案失败了，因为在网页里面的php代码不能执行git pull，有说是权限问题的，我尝试了几次不行就放弃了，因为对php不熟悉。

但是！我现在也实现了自动发布博文功能！是用Dropbox实现的，稍等我就来记载一下
