配置sublime

#1. Package Control
使用sublime3最方便的安装package的方法就是使用Package Control，Package Control安装起来还是很容易的，在sublime的控制台中输入一句话（这句话看起来有点长）就好了，控制台的快捷键是Ctrl + ~，也可以从菜单View - Show Console打开，打开之后复制如下一句话（注意这真的是一句话）：

import urllib.request,os; pf = 'Package Control.sublime-package'; ipp = sublime.installed_packages_path(); urllib.request.install_opener( urllib.request.build_opener( urllib.request.ProxyHandler()) ); open(os.path.join(ipp, pf), 'wb').write(urllib.request.urlopen( 'http://sublime.wbond.net/' + pf.replace(' ','%20')).read())

如果发现这种简单的方法不能用的话可以试试手动安装的方法，方法连接在这里。

安装好之后可以打开sublime的命令模式（Ctrl + Shift + P），输入Package Control就可以看到Package Control的菜单了，可以在里面安装package，也可以删除，如果只是想安装package的话可以输入install作为关键字，回车之后就可以看到网络上的sublime3的各种package了。

#2. Python
当时买sublime的第一原因就是想要找个写python的IDE，然后配置一通下来之后发现sublime无愧于神器之称，充分的满足了程序员们想要敲命令的癖好……

##2.1 Anaconda
这里说的Anaconda不是那个python的集成科学计算库，而是sublime的一个package，使用Anaconda可以方便的进行命令补全，而且可以自动的给你的python代码进行pep8格式化，那么安装Anaconda非常简单，只需要在选择package control的安装功能后输入Anaconda进行搜索，然后点击全字匹配的那个就好了，package control会帮你把Anaconda安装好的。

然后的问题就是把Anaconda配置的让自己喜欢就好了。
